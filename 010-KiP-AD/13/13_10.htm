Пульт управления электронным регулятором двигателя ЭРД-3ВМ размещен на центральном пульте кабины пилотов по крышкой из оргстекла.

На пульте расположены:

  • два выключателя ЭРД, предназначенные для включения-выключения ЭРД каждого двигателя;
  • два вылючателя "ЧР", которые служат для приведения в готовность канала ЧР соответствующего двигателя;
  • два нажимных переключателя "СТ1-РАБ-СТ2" для контроля АЗСТ каждого двигателя;
  • нажимной переключатель "ТК ПРАВ-РАБ-ТК ЛЕВ" для контроля контура ТК каждого двигателя.
